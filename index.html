Project Description: Dockerixed website deployment

Designed and deployed a static website using Docker containers to ensure consistent development and deployment environments. Implemented automation using bash scripting for application build, container management and deployment tasks. Utilized Git for version control and Github for source code management and collaboration. Build Docker images and published them to Dockerhub for easy distribution and deployment. Automated the deployment workflow to streamline application delivery and reduce manual effort

Technologies Used:

Docker, Bash Scripting, Git, Github, Dockerhub, linux
